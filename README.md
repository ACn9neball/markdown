# DevDeck

---

## 📱 Basic Description

Simple TUI for both saving information on personal projects and automated readmes based on the project information

---

## 🔍 Detailed Description

This application is a keyboard-driven terminal dashboard designed to streamline personal developer workflows by acting as both a local repository
tracker and an automated documentation engine. Operating entirely within the command line, it eliminates project management friction by allowing you
to catalog your active codebases, tech stacks, and features in a single, centralized view. The core utility lies in its markdown compilation engine:
with a single keystory, the tool extracts your saved metadata, selects a tailored layout template, and dynamically injects standardized installation
steps, prerequisites, and dynamic status badges to instantly generate a professional README.md right in your project's root directory.

---

## 🚀 Features


* **Core Feature 1:** Readme Creation
* **Core Feature 2:** Add project metadata
* **Core Feature 3:** View all projects
* **Core Feature 4:** Edit project metadata
* **Core Feature 5:** Search functionality
